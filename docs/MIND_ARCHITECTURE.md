# Mind Architecture

## Layer Definitions

### UC (Unconscious Processor)
Background processing. Pattern detection, filtering, anomaly flagging.
Runs continuously. Outputs feed into STM.

### STM (Short-Term Memory)
Active working memory. Current context, active reasoning, pending decisions.
Capacity-limited. Decays without reinforcement.

### LTM (Long-Term Memory)
Durable compressed knowledge. Patterns, relationships, stories, values.
Written via Learning Transfer Gate only.

### Decision Formation
Combines STM context + LTM patterns to generate candidate actions.
Outputs Decision contract — never executes directly.
