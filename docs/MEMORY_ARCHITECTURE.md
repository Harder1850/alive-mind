# Memory Architecture

## Experience Stream (ES)
Read-only input from Runtime. Raw perceptions before processing. Feeds UC layer.

## STM (Short-Term Memory)
Active reasoning context. Max entries defined in Constitution invariants. Decays or compresses to LTM.

## LTM (Long-Term Memory)
Compressed durable knowledge. Written only via Learning Transfer Gate (LTG).

## UC (Unconscious)
Background processing. Pattern detection, filtering. Pre-attentive screening before STM.
